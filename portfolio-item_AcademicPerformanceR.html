Student Academic Performance Analysis

  1. Portfolio
  2. Project 4

Project Details

  • Analyzed dataset based on student Academic Performance to find the top personal, professional and social factors which have an influence on student's final grades
  • Performed data preprocessing which included data cleaning using R and XLMiner to handle missing values, outliers in the dataset,and data transformation to correctly identify the type of variables used for analysis
  • Built three predictive machine learning models- multiple linear regression model, Decision Tree model, and Random Forest model to predict student's grades and compared model accuracy by RMSE
  • Determined factors influencing the target variable using forward and backward variable selection methods
  • Variable Importance plot was designed using Random Forest model to determine top 10 factors which had highest influence on the student's grades
  • Designed a presentation which included insights from the analysis, graphs, and charts for better understanding and recommendations for improving grades
