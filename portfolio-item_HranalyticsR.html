Human Resource Employee Attrition- R

  1. Portfolio
  2. Project 1

Project Details

  • Analyzed the problem of employee attrition in Human Resource department dataset consisting of 1470 rows and 35 columns in R programming
  • Visualized different contributing factors towards attrition by using ggplot package and Power BI
  • Performed Factor Analysis, used backward elimination method to find top contributing factors towards attrition
  • Built three machine learning models namely - Logistic Regression model, Random Forest model, and Decision Tree model to train the data set and predict results
  • Compared accuracy of different machine learning models by making confusion matrix
    • Accuracy for Logistic regression model: 89.6%
    • Accuracy for Decision Tree model:85.8%
    • Accuracy for Random forest model: 85.01%
